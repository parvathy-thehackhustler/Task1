# Task 1: Data Cleaning & Preprocessing

## Objective
Clean and prepare raw data for machine learning using Python.

## Tools Used
- Python
- Pandas, NumPy
- Seaborn, Matplotlib
- Jupyter Notebook

## Steps Performed
1. Imported the Titanic dataset and explored data types & null values.
2. Handled missing values (used median for Age, mode for Embarked).
3. Encoded categorical features (`Sex` and `Embarked`) using one-hot encoding.
4. Normalized numerical features using `StandardScaler`.
5. Visualized outliers using boxplots and removed them using the IQR method.

## Files Included
- `task1_data_cleaning.ipynb` — Code notebook
- `titanic.csv` — Raw dataset
- `README.md` — This file
- `screenshot1.png` — Boxplot visualization (optional)

## Output
A cleaned dataset ready for ML model training.
